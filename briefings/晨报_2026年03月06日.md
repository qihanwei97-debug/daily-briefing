### 🔥 头条：OpenAI发布GPT-5.4——首个内置Computer Use的主力模型

3月5日，OpenAI发布GPT-5.4系列，包含标准版、GPT-5.4 Thinking（推理增强版）和GPT-5.4 Pro（高性能版）。这是OpenAI迄今最强的基础模型，也是首个原生集成Computer Use能力的主力模型——Agent可直接操作软件界面，执行"构建-运行-验证-修复"闭环。

**核心技术指标：**
- **上下文窗口：** API版支持最高**1M tokens**（OpenAI首次达到此规模，追平Gemini 3.1 Pro）
- **GDPval（知识工作）：** 83.0%（GPT-5.2为70.9%，提升17%）
- **OSWorld Verified（桌面操作）：** 75.0%（GPT-5.2为47.3%，**首次超越人类水平72.4%**）
- **错误率：** 单条声明错误降低33%，整体回复错误降低18%
- **Token效率：** 解决同等问题所需token显著减少；Tool Search功能测试中节省47% token消耗
- **电子表格建模（投行分析师级任务）：** 87.3%（GPT-5.2为68.4%）

**定价（$/M tokens）：**
- GPT-5.4标准版：$2.50 input / $15 output
- GPT-5.4 Pro：$30 input / $180 output
- 超过272K input tokens后，input 2x、output 1.5x
- Batch/Flex模式半价，Priority模式双倍

**📍 获取：** ChatGPT（Plus/Teams/Pro用户）、Codex、API均已上线，model name `gpt-5.4`
**💰 定价：** 标准版$2.50/$15，对比Claude Opus 4.6的$15/$75，**输入便宜6倍、输出便宜5倍**
**💡 上手：** Computer Use + Tool Search是核心差异化——适合需要Agent自动操作软件、处理电子表格等企业级工作流
