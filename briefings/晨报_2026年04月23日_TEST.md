**`/ultrareview` 怎么用：** 运行后 Claude 自动拆成多个 Agent（安全/架构/测试/可读性），各自独立审查当前 branch 或指定 PR，最后合并输出。**律师合同审阅的直接映射**：拆成法律风险 / 条款遗漏 / 引用核验 / 语言精度四个视角并行检查。

**为什么重要：** Claude Code 从 CLI 工具演进为"AI 操作系统"——`/ultrareview` 是 **AI 替代 Junior 审阅**的明确信号。对大所而言是增量工具（提速），不是替代。

- 📍 获取：`claude --update` | [changelog](https://code.claude.com/docs/en/changelog)
- 💰 xhigh 需 Max 订阅（$100/月）；`/ultrareview` 单次约 $0.50-1.50
- 💡 上手：把合同放 repo → `claude` → `/ultrareview` → 观察多视角输出

## ⑤ 实用技巧

**`/ultrareview` 多 Agent 审查**（`Claude Code`）
- 🎯 开发者 PR review；律师合同/诉状多视角并行审阅
- 🔧 `claude` → `/ultrareview`（审当前 branch）或 `/ultrareview <PR#>`
- ⚠️ Max 订阅 + Opus 4.7 才能用；单次 $0.50-1.50；合同需先 `git init`
- 🔗 [changelog v2.1.111](https://code.claude.com/docs/en/changelog)

**1 小时 prompt cache 省钱**（`工作流`）
- 🎯 长时间法律研究/代码重构会话——避免每次重新计费 system prompt
- 🔧 Windows：`setx ENABLE_PROMPT_CACHING_1H 1` → 重启 claude
- ⚠️ 长会话省 40-60%；Opus cache rate $1.50/M vs 正常 $15/M
- 🔗 [changelog v2.1.108](https://code.claude.com/docs/en/changelog)

**`/tui` 解决 Windows 闪屏**（`踩坑警示`）
- 🎯 Windows Terminal 跑 Claude Code 时闪烁/光标跳动
- 🔧 会话内输入 `/tui` 即可
- ⚠️ 仅当前会话生效
- 🔗 [changelog v2.1.110](https://code.claude.com/docs/en/changelog)
