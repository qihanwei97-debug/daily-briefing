为什么重要：
  • AI 算力市场已"信用卡化"——少数 frontier 厂商单合同就锁定 hyperscaler 五年容量的一半以上——这意味着 mid-tier 企业客户在 2027-2030 年想批量采购 TPU/GPU 算力会面临实质性挤出，要么排队、要么转向 specialized 二线供应商（CoreWeave/Lambda/Nebius/Crusoe）；这与 Big Tech Q1 财报上调 Capex（5/6 晨报头条二）形成完整闭环：hyperscaler 加 Capex 是为了消化已签长约，不是机会主义扩产
  • Anthropic 算力组合"AWS + Google + Azure + SpaceX + Fluidstack"全景与 OpenAI 形成对照——OpenAI 是"Azure + Oracle + Stargate（OpenAI/SoftBank/Oracle JV）"的相对集中，Anthropic 走分散化；这是 AI 投资人估值两家时的核心结构差异——分散降单点风险，集中保议价能力，难以一概而论
  • $350B Google 投资估值确认——Anthropic 在 OpenAI（$500B 估值传闻）之后稳坐第二把交椅；按 Anthropic 公开 ARR 测算（$8-10B 年化），P/S 倍数约 35-44x，仍属 frontier 厂商区间
  • TPU 上量节奏——Google "multi-GW" 容量 2027 起交付意味着 2026 全年 Anthropic 主要靠 AWS Trainium + 现有 Google TPU + SpaceX Colossus 1 顶住——SpaceX 220K GPU 是"近月"补丁、Google TPU 是"长周期"主干

本期技巧

用 Claude × Microsoft 365 单 agent 跨 Excel/Word/PPT/Outlook 上下文打通做合同审阅
