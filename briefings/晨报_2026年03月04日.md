## 二、AI 深度报道

### 头条：NVIDIA 40亿美元押注光子互联，重塑AI基础设施供应链

NVIDIA 于3月2日宣布分别向光子芯片公司 Lumentum 和 Coherent 各投资20亿美元，总计40亿美元。这不仅是财务投资——还包含数十亿美元的多年期采购承诺和未来产能优先权。NVIDIA 的战略意图清晰：从单纯的GPU供应商，向掌控整个AI数据中心基础设施（算力→网络→光互联组件）的方向转型。光子芯片是下一代数据中心内部高速通信的核心，传统铜缆在AI集群的带宽和能耗瓶颈日益明显。消息公布后 Lumentum 股价涨11.75%，Coherent 涨15.44%。（[NVIDIA Newsroom - Lumentum, 2026-03-02](https://nvidianews.nvidia.com/news/nvidia-announces-strategic-partnership-with-lumentum-to-develop-state-of-the-art-optics-technology); [NVIDIA Newsroom - Coherent, 2026-03-02](https://nvidianews.nvidia.com/news/nvidia-and-coherent-announce-strategic-partnership-to-develop-optics-technology-to-scale-next-generation-data-center-architecture); [CNBC, 2026-03-02](https://www.cnbc.com/2026/03/02/nvidia-investment-coherent-lumentum.html)）

- 💡 **上手提示**: 关注光子互联（silicon photonics）赛道的投资机会，以及对AI算力成本长期下降的影响

### 头条：Apple March Event 今日揭幕——M5芯片、MacBook Neo 与 AI Siri 新进展

Apple 于3月2-4日在纽约、伦敦、上海同步举办"Special Apple Experience"，今日为正式发布日。确认发布的产品包括：搭载 M5 芯片的 MacBook Air、搭载 M5 Pro/M5 Max 的 MacBook Pro、全新低价入门级 MacBook Neo（售价或低于$999）、两款新 Studio Display、以及 iPhone 17e（3月4日开始预购，3月11日发售）。iPhone 17e 首次升级至 A19 处理器并新增 MagSafe 支持。M5 芯片据传 GPU 性能提升30-45%，每个 GPU 核心集成 Neural Accelerator 以加速AI工作负载。Siri AI 升级方面，基础版 Gemini 驱动的 Siri 预计随 iOS 26.4 推送，但高级功能（跨应用上下文理解）可能延期至 iOS 26.5 或 iOS 27。（[MacRumors, 2026-03](https://www.macrumors.com/guide/apple-products-march-2026/); [Tom's Guide, 2026-03-04](https://www.tomsguide.com/news/live/apple-march-event-2026-live); [Creative Bloq, 2026-03-04](https://www.creativebloq.com/live/news/apple-event-march-2026)）

- 📍 **获取**: apple.com，iPhone 17e 今日开放预购
- 💰 **定价**: MacBook Neo 预计 <$999；iPhone 17e 预计 $599 起
- 💡 **上手提示**: M5 芯片的 Neural Accelerator 对本地AI推理（如 Stable Diffusion、Whisper）有显著加速，值得关注开发者实测

### 速报

**DeepSeek V4 发布倒计时** — DeepSeek 首个多模态旗舰模型 V4 预计本周正式发布，支持文本、图片、视频生成，已针对华为 Ascend 和寒武纪芯片优化。两会开幕为发布增添政策背景。（[Yahoo News, 2026-03](https://malaysia.news.yahoo.com/deepseek-release-long-awaited-ai-162425050.html); [PYMNTS, 2026-03](https://www.pymnts.com/artificial-intelligence-2/2026/deepseek-poised-to-unveil-latest-ai-model/)）

- 📍 **获取**: 预计通过 chat.deepseek.com 及 API 同步开放

**MiniMax M2.5 持续搅动市场** — 2月12日发布的 MiniMax M2.5（230B MoE，10B 活跃参数）在 SWE-Bench Verified 达到 80.2%，多语言编程基准 Multi-SWE-Bench 行业第一，定价仅为 Claude Opus 4.6 的 1/20（输入$0.29/M，输出$1.20/M）。MiniMax 内部已有30%任务由 M2.5 完成，80%新代码由 M2.5 生成。（[VentureBeat, 2026-02](https://venturebeat.com/technology/minimaxs-new-open-m2-5-and-m2-5-lightning-near-state-of-the-art-while); [MiniMax 官网, 2026-02](https://www.minimax.io/news/minimax-m25)）

- 📍 **获取**: minimax.io, API 开放
- 💰 **定价**: 输入 $0.29/M tokens，输出 $1.20/M tokens
- 💡 **上手提示**: 编程和 Agent 场景下性价比极高，可作为 Claude/GPT 的低成本替代

**美国考虑收紧对华AI芯片出口** — 美国官员正考虑对单一中国买家设定 Nvidia H200 级AI加速器的出口上限，以阻断中国大型AI实验室集中采购高端芯片的路径。（[搜索综合, 2026-03]）

**AI 推理安全成为2026年企业焦点** — 专家指出，AI安全讨论过度集中在训练阶段，而推理阶段（模型被查询使用时）的安全风险被严重低估——专有模型逻辑和敏感提示词在此阶段暴露。调查显示46.2%的企业表示对现有AI系统满足2026安全标准"没有信心"。（[The Quantum Insider, 2026-03-03](https://thequantuminsider.com/2026/03/03/securing-ai-inference-the-overlooked-security-frontier-in-2026/)）
